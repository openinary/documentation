---
title: "Local Development"
description: "Guide for developing Openinary locally"
---

## Start Everything

```bash
pnpm dev
```

This will start both the API and Frontend simultaneously.

## Start Separately

```bash
pnpm dev:api
```

```bash
pnpm dev:web
```

## Build all

```bash
pnpm build
```

## Lint all

```bash
pnpm lint
```
